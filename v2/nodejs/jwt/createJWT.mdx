# `createJWT(payload, validitySeconds)`

### Parameters
- `payload` (Optional)
  - type: `any`
  - Must be a valid JSON

- `validitySeconds` (Optional)
  - type: `number`
  - Must be greater than 0

### Returns
```ts
Promise<{
    status: "OK",
    jwt: string,
} | {
    status: "UNSUPPORTED_ALGORITHM_ERROR",
}>
```
