# ``updateEmailOrPassword(input: {userId: string, email?: string, password?: string})``

### Parameters
- ``userId``
  - type: ``string``
- `email` (Optional)
  - type: `string`
  - This is the new email to update to.
- `password` (Optional)
  - type: `string`
  - This is the new password to update to.


### Returns
```ts
Promise<{
    status: "OK" | "EMAIL_ALREADY_EXISTS_ERROR" | "UNKNOWN_USER_ID_ERROR";
}>
```
