# Disabling APIs

<BackendSDKTabs>
<TabItem value="nodejs">

To disable an API entirely, all you need to do is override the api implementation with `undefined`.

For example, if you want to disable the sign-up / sign-in api from this recipe, all you do is this:

```jsx
SuperTokens.init({
    appInfo: {...},
    supertokens: {...},
    recipeList: [
        ThirdPartyEmailPassword.init({
            // highlight-start
            override: {
                apis: (originalImplementation) => {
                    return {
                        ...originalImplementation,
                        signInUpPOST: undefined
                    }
                }
            }
            // highlight-end
        })
    ]
});
```
</TabItem>
<TabItem value="go">

To disable an API entirely, all you need to do is override the api implementation with `nil`.

For example, if you want to disable the sign-up / sign-in api from this recipe, all you do is this:

```go
func main() {
	supertokens.Init(supertokens.TypeInput{
		RecipeList: []supertokens.Recipe{
			thirdpartyemailpassword.Init(&tpepmodels.TypeInput{
				//highlight-start
				Override: &tpepmodels.OverrideStruct{
					APIs: func(originalImplementation tpepmodels.APIInterface) tpepmodels.APIInterface {

						//First we copy the original impl
						newImplementation := originalImplementation

						// we set the sign in / up API to nil to disable it
						newImplementation.SignInUpPOST = nil

						return newImplementation
					},
				},
				//highlight-end
			}),
		},
	})
}
```
</TabItem>
<TabItem value="python">

To disable an API entirely, all you need to do is override the api disable bool value to `True`.

For example, if you want to disable the thirdparty sign-up / sign-in api from this recipe, all you do is this:

```python

# highlight-start
def apis_override(param):
    param.disable_thirdparty_sign_in_up_post = True
    return param
# highlight-end

supertokens_python.init({
    'app_info': {...},
    'supertokens': {...},
	'framework': '...',
    'recipe_list': [
        thirdpartyemailpassword.init({
            # highlight-start
            'override': {
                'apis': apis_override
            }
            # highlight-end
        })
    ]
})
```

</TabItem>
</BackendSDKTabs>
