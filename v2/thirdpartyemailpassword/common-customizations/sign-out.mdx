# Sign out

<FrontendSDKTabs>
<TabItem value="reactjs">

The [`signOut` method](https://supertokens.com/docs/auth-react/modules/recipe_^{docsLinkRecipeName}.html#signOut) revokes the session on the frontend and backend.

```tsx
import React from "react";
// highlight-next-line
import { signOut } from "supertokens-auth-react/recipe/^{codeImportRecipeName}";

function NavBar() {
  async function onLogout() {
    // highlight-next-line
    await signOut();
    window.location.href = "/";
  }
  return (
    <ul>
      <li>Home</li>
          // highlight-next-line
      <li onClick={onLogout}>Logout</li>

    </ul>
  )
}
```
</TabItem>

The [`signOut` method](https://supertokens.com/docs/website/modules.html#signOut) revokes the session for the user.

```tsx
// highlight-next-line
import SuperTokens from "supertokens-website";

async function logout () {
  // highlight-next-line
  await SuperTokens.signOut(); 
  window.location.href = "/";
}
```
