# EmailPassword and Social (Third party / OAuth 2.0 / SAML) login with SuperTokens

- Sign-up / Sign-in with email and password or with OAuth 2.0 (Like google, facebook etc..) and SAML providers <br/>
- Forgot password flow using email <br/>
- Secure session management <br/>
- Email verification <br/>
