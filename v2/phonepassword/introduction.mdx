# Phone number + Password login with SuperTokens

## Features
- Login and signup with phone number and password
- Phone number verification using OTP
- Secure session management
- Customise SMS
- Integrate with your own SMS sending service

## Demo application
We have a working demo app built using reactJS and NodeJS. Find the [source code on our GitHub](https://github.com/supertokens/supertokens-auth-react/tree/master/examples/with-phone-password).
