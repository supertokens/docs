# Passwordless login with SuperTokens

## Features

- Sign in / up with OTP or / and magic link via SMS or email <br/>
- Secure session management <br/>
- Customise email or SMS <br/>
- Integrate with your own email / SMS sending service <br/>

## Demo application

- See our [live demo app](https://emailpassword.demo.supertokens.com/)
- Generate a starter app
    ```bash
    npx create-supertokens-app --appname=my-app --recipe=passwordless
    ```
