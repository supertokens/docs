# ``updateEmailOrPassword(input: {userId: string, email?: string, password?: string})``

### Parameters
- ``userId``
  - type: ``string``
- `email` (Optional)
  - type: `string`
  - This is the new email to update to.
- `password` (Optional)
  - type: `string`
  - This is the new password to update to.


### Returns
- `Promise<void>`

### Throws
- If the user does not exist or did not sign up with emailpassword recipe
- If the provided email already belongs to another user.
