import {PreBuiltOrCustomUISwitcher, PreBuiltUIContent, CustomUIContent} from "/src/components/preBuiltOrCustomUISwitcher"

When using the `PreBuiltUIContent` or `CustomUIContent` component, headers only show in the right navbar when the specific UI type is selected by the user.

For example:

<PreBuiltOrCustomUISwitcher>

## Common Header

<PreBuiltUIContent>

## This header only shows for Pre built UI

</PreBuiltUIContent>

<CustomUIContent>

## This header only shows for Custom UI

</CustomUIContent>

</PreBuiltOrCustomUISwitcher>

```
<PreBuiltOrCustomUISwitcher>

## Common Header

<PreBuiltUIContent>

## This header only shows for Pre built UI

</PreBuiltUIContent>

<CustomUIContent>

## This header only shows for Custom UI

</CustomUIContent>

</PreBuiltOrCustomUISwitcher>
```
