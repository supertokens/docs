When using the `PreBuiltOrCustomUISwitcher` component, you can freely use any normal markdown feature or component that you normally use. Content inside of `PreBuiltUIContent` or `CustomUIContent` components will be selectively visible, content outside of these components will always be visible.

You can have multiple `PreBuiltUIContent` or `CustomUIContent` components in the tree.

## Regular Markdown

:::info
Tooltips
:::

### Headers

<PreBuiltUIContent>

## Pre Built Content

Content that applies only to pre built UI should use `PreBuiltUIContent`.

</PreBuiltUIContent>

<CustomUIContent>

## Custom Content

Content that applies only to custom UI should use `CustomUIContent`.

</CustomUIContent>

## Some content that is common to both

You can freely place items between multiple instances of `PreBuiltUIContent` and `CustomUIContent`, they will always be visible.

<PreBuiltUIContent>

## Some More Pre Built Content

Content that applies only to pre built UI should use `PreBuiltUIContent`.

</PreBuiltUIContent>

<CustomUIContent>

## Some More Custom Content

Content that applies only to custom UI should use `CustomUIContent`.

</CustomUIContent>
