# Using pre built and custom UI

## Separating content

You can use the `PreBuiltOrCustomUISwitcher` component as a wrapper for all your content in the mdx file

```mdx example.mdx
import {PreBuiltOrCustomUISwitcher} from "/src/components/preBuiltOrCustomUISwitcher"

<PreBuiltOrCustomUISwitcher>

# Some content goes here

</PreBuiltOrCustomUISwitcher>
```

You can use the `PreBuiltUIContent` and `CustomUIContent` to separate content based on what the user selects

```mdx
import {PreBuiltOrCustomUISwitcher, PreBuiltUIContent, CustomUIContent} from "/src/components/preBuiltOrCustomUISwitcher"

<PreBuiltOrCustomUISwitcher>

<PreBuiltUIContent>

Content for Pre built UI

</PreBuiltUIContent>

<CustomUIContent>

Content for Custom UI

</CustomUIContent>

</PreBuiltOrCustomUISwitcher>
```
